About OptiCrop

OptiCrop is a Machine Learning-based crop recommendation system that helps farmers select the most suitable crop based on soil nutrients and environmental conditions. The system analyzes Nitrogen (N), Phosphorus (P), Potassium (K), temperature, humidity, pH, and rainfall to predict the best crop for cultivation.

The project uses Python, Pandas, NumPy, Scikit-learn, Flask, HTML, and CSS. Multiple machine learning models were trained and evaluated, and the best-performing model was selected to provide accurate crop recommendations.
